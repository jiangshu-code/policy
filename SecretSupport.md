## 常见问题

### 免费内容与解锁

**Q1：不购买可以看什么？**  
免费用户可以看到 **3 条**随机示例秘诀，覆盖不同分类，帮你判断内容是否符合你的口味。分类页会显示每个分类的总条数与你当前可见的条数，例如 `1/9`。

**Q2：免费看到的 3 条内容会变吗？**  
**不会。** 首次打开 App 时抽签结果会被安全保存在系统 Keychain 中，即使**卸载 App 并重装**，看到的 3 条内容也保持不变。这是为了给用户稳定的示例体验。

**Q3：为什么免费示例里一定有一条"省钱"？**  
省钱是最直观、最能让人立刻感受到价值的分类，我们希望让第一次接触 App 的你就能获得一条能马上用起来的秘诀。

### 购买与恢复购买

**Q4：付费方式是什么？**  
一次性买断 **¥38**，永久解锁全部秘诀。**不是订阅**，不会自动续费，未来 App 内新增的秘诀也**免费**赠送给已购买用户。

**Q5：如何购买？**  
点击首页顶部或分类页的"解锁"按钮进入付款页，点击"立即解锁 ¥38" → 系统弹出 Face ID / Touch ID / 密码确认 → 支付完成后立即解锁所有内容。

**Q6：换新手机后，之前买的还能用吗？**  
**可以。** 用同一个 Apple ID 登录新设备，打开秘诀 App → 进入付款页 → 点击底部"**恢复购买**"按钮，App 会向 Apple 服务器验证你的购买记录，几秒内自动恢复解锁状态。

**Q7："恢复购买"没反应怎么办？**  
1. 确认当前登录的 Apple ID 与最初购买时**一致**
2. 检查是否有网络连接
3. 到 **设置 → [你的名字] → 媒体与购买项目 → 查看账户 → 购买历史记录**，确认购买记录存在
4. 若确认无误仍无法恢复，请联系我们，附上 Apple ID 邮箱和购买时间

**Q8：可以退款吗？**  
根据 Apple App Store 政策，退款由 Apple 统一处理，作者无权发起。你可以到 [reportaproblem.apple.com](https://reportaproblem.apple.com) 申请退款，通常 24–48 小时内会有回复。

### 跨设备与卸载重装

**Q9：iPhone 和 iPad 上买一次要买两次吗？**  
**不需要。** 用同一 Apple ID 登录任一设备，安装秘诀 App 后进入解锁页点击"恢复购买"即可。

**Q10：卸载 App 后再安装，需要重新付费吗？**  
**不需要。** 打开新装的 App → 解锁页 → "恢复购买"即可。你的付费记录保存在 Apple 服务器，不受本地卸载影响。

**Q11：卸载重装后，我看过的免费示例内容会变吗？**  
不会。抽签结果保存在系统 Keychain，跨卸载依然有效。（详见 Q2）

**Q12：更换 Apple ID 会失去解锁状态吗？**  
会。付费记录绑定 Apple ID，切换到另一个未购买的 Apple ID 后需要用原账号才能恢复。这是 Apple 平台的统一规则，作者无法绕开。

### 内容与更新

**Q13：秘诀内容会持续更新吗？**  
会。作者会持续基于新的经验与读者反馈补充内容。**所有更新对已购买用户免费**。

**Q14：如何知道有新内容？**  
每次 App 版本更新时，App Store 的"更新说明"会列出本次新增的秘诀数量与所属分类。你也可以在 iOS **设置 → App Store → App 自动更新**开启自动更新，第一时间获取新内容。

**Q15：能建议新增某个主题的秘诀吗？**  
非常欢迎！请通过下方[联系我们](#联系我们)反馈。若你的建议被采纳，会在版本更新说明中致谢。

**Q16：内容准确性如何保证？**  
所有秘诀基于作者的亲身经验或长期观察。**但请注意**：涉及法律、医疗、财务、政策等领域的内容仅供参考，具体决策请以当地法规和专业意见为准。App 不构成任何专业建议。

### 隐私与数据

**Q17：App 会收集我的个人信息吗？**  
**不会。** 秘诀 App 不收集、不上传、不存储任何用户身份信息、行为数据、位置或联系人。所有本地数据（免费示例抽签结果、引导页已读标记等）仅存在你自己的设备上。

**Q18：付款信息安全吗？**  
所有付款均通过 **Apple StoreKit** 完成，作者永远看不到你的支付信息（银行卡、Apple ID 密码等）。这些数据受 Apple 隐私政策保护。

**Q19：完整隐私政策在哪里？**  
参见 [隐私政策](./PrivacyPolicy.md)。

## 故障排查

### App 无法启动 / 闪退
1. 强制退出 App：上滑呼出后台 → 上滑关闭秘诀
2. 重启设备
3. 更新到最新版 iOS（**设置 → 通用 → 软件更新**）
4. 到 App Store 检查秘诀是否有可用更新
5. 仍无法解决，请卸载重装（付费和示例记录不会丢失）

### 支付按钮点击无反应
1. 确认设备已联网
2. 检查 **设置 → [你的名字] → 媒体与购买项目** 是否已登录
3. 检查是否开启了"**屏幕使用时间 → 内容和隐私访问限制 → App 内购买**"限制
4. 尝试重启 App

### 解锁后仍显示未解锁
1. 完全退出 App 后重新打开
2. 到解锁页点击"恢复购买"
3. 检查是否有多个 Apple ID 混用的情况
4. 联系我们，附上 Apple ID 邮箱和交易 ID

### 图标 / 文字显示异常
1. 检查 iOS 版本是否 ≥ 15.1（部分 SF Symbols 需要更高版本）
2. 尝试切换深色 / 浅色模式
3. 重启 App
